# Wifi Password Defaults

### `WiFi-0000`
Characters: 0-9
Length: 8
Example: 12345678

### `NETGEAR00`
Format: nounadjective0-9
Example: zooyoung123

### `MySpectrumWiFiA0`
Format: noun adjective 0-9
Example: zooyoung123

### `TelstraABC123`
Characters: A-Z+0-9
Length: 10
Example: A0B1C2D3E4

### `TelstraAB12`
Characters: 0-9
Length: 10
Example: 1234567890

### `OPTUSVDABCD123`
Format: noun (5char upper), noun (5char upper), 5 num
Example: HORSESTUFF12345

### `OPTUS_ABC123`
Format: noun (5char lower), noun (5char lower), 5 num
Example: cakeyflimp12345

### `Optus-4G-AB12-CD34`
Characters: A-Z 0-9
Length: 11
Example: 1A2B3C4D5E6
